# ArtCert Team 2

Hello! 

This repository serves as the central documentation and development hub for Team 2’s Masters Software Engineering Project (SWEN90014). The ArtCert Project is designed to provide a secure and trusted platform where artists and collectors can authenticate, trade, and manage verified artworks on a global scale.

Within this repository, we outline our team structure, provide release notes, and maintain a detailed changelog to ensure transparency and traceability throughout the project’s development lifecycle.

## Executable Architecture (UPDATED)

Executable architecture is a software architecture model that is not just documented but also implemented and runnable early in the development process. It provides a working skeleton of the system — demonstrating core components, interactions, and critical design decisions so that architectural choices can be validated, tested, and refined through execution.
